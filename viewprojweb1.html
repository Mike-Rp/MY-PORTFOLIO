About This Project

The Affiliate Landing Page is a fully responsive and modern static website designed to attract and engage users interested in affiliate programs. The page focuses on visual clarity and persuasive design to guide visitors toward taking action and joining as affiliates.

This project involved designing the user interface and experience in Figma and developing the site using HTML, Tailwind CSS, and JavaScript. The responsive layout ensures optimal display across all screen sizes, while smooth transitions and structured sections enhance overall usability.

Key features include clear call-to-action sections, scroll-based navigation, and fast-loading performance optimized for both desktop and mobile. The website was coded and deployed by me and is live for public access via GitHub Pages.
