About This Project

This is best for those who proscastinate a lot and have trouble focusing on their work. The Alarm Task website provides a simple and effective solution by allowing users to set alarms that will disable sound on their monitors, helping them stay focused and avoid distractions.

Recommended for those who have dual monitor, as it allows users to mute one monitor while keeping the other active for work-related tasks.

The website features a clean and intuitive interface designed using Figma, ensuring a seamless user experience. Built with HTML, CSS, and Tailwind CSS, the site is fully responsive and optimized for performance across all devices.
